ABAP

```
1、所有提交皆由分支（master/main），所有发布皆由主干（master）

2、查看分支信息
git branch -all (远程分支）
git branch      (本地分支）

3、切换分支
git checkout （master/main）

4、提交并推送分支
git add .
git commit -m  "描述"
git push -u origin （master/main）

5、本地分支合并
git merge （master/main）

6、推送主分支
git push -u origin （master/main）
 

```
